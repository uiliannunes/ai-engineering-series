🦜🔗LangChain na Prática

Seu Primeiro Prompt Template — do zero ao pipeline funcional

PromptTemplate ChatOpenAI LLMChain chain.invoke()

🤔 O que é LangChain?

LangChain é um framework Python que facilita a criação de aplicações com LLMs (Large Language Models), como GPT, Claude ou Gemini.

Em vez de montar chamadas de API na mão e gerenciar strings de prompt manualmente, o LangChain organiza tudo em blocos reutilizáveis e encadeáveis.

❌ Sem LangChain:  prompt (string) → chamada API → parse da resposta  trabalhoso

✅ Com LangChain:  PromptTemplate → LLM → OutputParser  organizado

✍️ Passo 3 — Criando o PromptTemplate

O PromptTemplate permite criar prompts com variáveis dinâmicas usando {nome_da_variavel}.

Isso é muito útil quando você quer reutilizar o mesmo prompt com entradas diferentes.

[3]
python 
from langchain.prompts import PromptTemplate

# Definindo o template com uma variável {produto}
template = PromptTemplate(
    input_variables=["produto"],
    template="Escreva um slogan criativo para o produto: {produto}"
)

# Visualizando o prompt formatado antes de enviar ao modelo
prompt_formatado = template.format(produto="tênis esportivo")
print("📝 Prompt gerado:")
print(prompt_formatado)

💡 Dica: Templates com múltiplas variáveis

Você pode ter quantas variáveis quiser no mesmo template:

[4]
python 
template_avancado = PromptTemplate(
    input_variables=["produto", "publico", "tom"],
    template="""
Você é um especialista em marketing.
Crie um slogan para o produto: {produto}
Público-alvo: {publico}
Tom de comunicação: {tom}
"""
)

print(template_avancado.format(
    produto="tênis esportivo",
    publico="jovens entre 18 e 30 anos",
    tom="energético e motivacional"
))

🤖 Passo 4 — Conectando ao Modelo e Executando

Agora vamos conectar o template ao modelo GPT e criar nosso primeiro pipeline completo.

PromptTemplate → ChatOpenAI → Resposta
[5]
python 
from langchain_openai import ChatOpenAI
from langchain.chains import LLMChain

# Inicializa o modelo
llm = ChatOpenAI(
    model="gpt-4o-mini",  # modelo mais econômico e eficiente
    temperature=0.7        # 0 = determinístico | 1 = mais criativo
)

# Conecta o template ao modelo em uma chain
chain = LLMChain(llm=llm, prompt=template)

# Executa o pipeline
resposta = chain.invoke({"produto": "tênis esportivo"})

print("🎯 Resposta do modelo:")
print(resposta["text"])

🔄 Passo 5 — Testando com múltiplos inputs

A reutilização é uma das grandes vantagens do PromptTemplate. Veja como gerar slogans para vários produtos com o mesmo pipeline:

[6]
python 
produtos = [
    "notebook gamer",
    "café especial artesanal",
    "aplicativo de meditação"
]

print("🚀 Gerando slogans...\n")
for produto in produtos:
    resposta = chain.invoke({"produto": produto})
    print(f"📦 {produto.upper()}")
    print(f"   → {resposta['text'].strip()}")
    print()

✅ Resumo do que você aprendeu

Conceito O que é
PromptTemplate Cria prompts dinâmicos com variáveis reutilizáveis
ChatOpenAI Conector para os modelos GPT da OpenAI
LLMChain Encadeia um template + modelo em um pipeline
chain.invoke() Executa o pipeline com os inputs fornecidos
