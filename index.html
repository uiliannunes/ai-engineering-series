🤔 O que é LangChain?

LangChain é um framework Python que facilita a criação de aplicações com LLMs (Large Language Models), como GPT, Claude ou Llama.

Em vez de montar chamadas de API na mão, o LangChain organiza tudo em blocos reutilizáveis e encadeáveis.

❌ Sem LangChain:  prompt (string) → chamada API → parse  trabalhoso

✅ Com LangChain:  PromptTemplate → LLM → OutputParser  organizado

✍️ Passo 3 — Criando o PromptTemplate

O PromptTemplate permite criar prompts com variáveis dinâmicas usando {nome_da_variavel}.

[3]
python
from langchain_core.prompts import PromptTemplate

template = PromptTemplate(
    input_variables=["produto"],
    template="Escreva um slogan criativo para o produto: {produto}"
)

prompt_formatado = template.format(produto="tênis esportivo")
print("📝 Prompt gerado:")
print(prompt_formatado)
output
📝 Prompt gerado:
Escreva um slogan criativo para o produto: tênis esportivo

🤖 Passo 4 — Conectando ao Modelo e Executando via Groq

Conecte o template ao modelo Llama 3.1 via Groq usando a sintaxe LCEL (LangChain Expression Language) com o operador |.

PromptTemplate | ChatGroq → response.content
[5]
python
from langchain_groq import ChatGroq
from langchain_core.prompts import PromptTemplate

template = PromptTemplate.from_template(
    "Crie uma descrição criativa para vender um {produto}"
)

llm = ChatGroq(
    model="llama-3.1-8b-instant",
    temperature=0.7
)

# Sintaxe LCEL: encadeia template e modelo com o operador |
chain = template | llm

response = chain.invoke({"produto": "tênis esportivo"})
print(response.content)
output
**Desbloqueie o seu potencial com o "Thunderbolt"!**

Você está pronto para dar um salto para o topo? Com o "Thunderbolt",
o seu tênis esportivo mais potente, você estará preparado para
enfrentar qualquer desafio que vier sua maneira...

🔄 Passo 5 — Testando com múltiplos inputs

Veja como gerar descrições para vários produtos com o mesmo pipeline:

[6]
python
produtos = [
    "notebook gamer",
    "café especial artesanal",
    "aplicativo de meditação"
]

print("🚀 Gerando slogans...\n")
for produto in produtos:
    resposta = chain.invoke({"produto": produto})
    print(f"📦 {produto.upper()}")
    print(f"  → {resposta.content.strip()}")  # .content em vez de ['text']
    print()
output

✅ Resumo do que você aprendeu

ConceitoO que é
PromptTemplateCria prompts dinâmicos com variáveis reutilizáveis
ChatGroqConector para modelos Groq (Llama, Mixtral etc) — gratuito
template | llmSintaxe LCEL para encadear template + modelo em um pipeline
chain.invoke()Executa o pipeline com os inputs fornecidos
response.contentAcessa o texto da resposta no formato LCEL
